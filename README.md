how to host:
install uwamp
https://www.uwamp.com/en/?page=download
set it up do whatever it tells you to do then go to www and unzip the site, inside the folder (or folders) there should be assets, gallery and etc, do CTRL A (grab them all) and then CTRL X (or move them) to www

you're done, just turn on the webserver and it should be running.
also just delete this readme once its fully setup.
